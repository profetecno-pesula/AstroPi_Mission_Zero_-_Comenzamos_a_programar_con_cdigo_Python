2. ¿Qué es un Astro Pi?

 

Las computadoras Astro Pi vienen con un conjunto de sensores y dispositivos que se pueden usar para realizar grandes experimentos científicos. 

Este conjunto de sensores se llama 'Sense HAT' (que significa 'Hardware adjunto en la parte superior'). El Sense HAT le da a Astro Pi la capacidad de 'detectar' y realizar muchos tipos de mediciones, desde la temperatura hasta el movimiento, y generar información utilizando una pantalla de matriz de LED de 8 x 8. 

Las Astro Pis también tienen un joystick y botones, ¡como una consola de videojuegos!
